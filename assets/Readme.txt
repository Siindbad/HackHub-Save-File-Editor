=================================================================================
              [ HACKHUB SAVE EDITOR ]
=================================================================================
DISCORD : snowbad66x1
FOR GAME : Hackhub
=================================================================================


		      [ USAGE ]
=================================================================================

1. In-Game : "F1" and type "save.export" will drop a save file
2. Create a "Backup" Folder where it dropped
3. Copy / Paste your save in a backup folder for safe measures
4. OPEN the save with Sin Editor
5. Make changes needed then "Apply Edit"
6. "Export" The save in same location or new, overwriting is fine
7. In-Game at MAIN MENU : Use F1 then type "save.import" and select the file

=================================================================================

Warning : This editor is good for changing simple things like bank account money
or Services that aren't showing, or even opening or even adding ports.

Changing to much can result in errors of the domain/IP : that's why its best to
always have a backup save incase you made errors.

Ensure to be very observant to the information you are looking at before making
any changes, and ensure you are within the right parameters to make that change.

=================================================================================

For Community Helpers : This tool helps pin down a persons bug in which you can
quickly assist them with credentials or opening bugged ports, or even obtaining
there HackTheCube web address if they deleted the email.
